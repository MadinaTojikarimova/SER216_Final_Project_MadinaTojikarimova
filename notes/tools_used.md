# Tools Used

## Google Docs
Used to write and format the full project report including all tasks, requirements, defect analysis, and submission summary.

## Google Sheets
Used to create the test case table with 6 test cases covering system, acceptance, and regression testing.

## diagrams.net (draw.io)
Used to design the workflow diagram showing the full process of the Digital Parking Permit System from vehicle registration to parking verification.

## GitHub
Used for version control throughout the project:
- Created repository to store all project files
- Created feature branch (feature/test-plan) to work separately from main
- Made commits to track changes and progress
- Opened and merged a pull request to combine work into main branch
- Used as final submission evidence
